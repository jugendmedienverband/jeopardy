# Jeopardy

> A jeopardy game written in React

## Install

```bash
npm install
```

## Edit game
See `src/game.js` for game data.

## Start a game

```bash
npm start
```

## Short cuts
* `t`: Enable or disable player input by Buzzer
* `r`: Select a random player
* `shift+r`: Deselect current player
* `s`: Show final screen
* `shift+<number>`: Player buzzer
